Installation:
``` shell
git clone https://github.com/abrdk/Scrawl
cd Scrawl
pip3 install -r requirements.txt
```

Start the server:
``` shell
flask run
```

Visit [127.0.0.1:5000](127.0.0.1:5000) in a browser.
