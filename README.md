# scrawl
Note-taking Software

Installation:
``` shell
git clone https://github.com/noooway/scrawl
cd scrawl
pip3 install -r requirements.txt
```

For testing, start server with:
``` shell
flask run
```

To make server visible to the outside world:
``` shell
flask run --host='0.0.0.0' --port='5000'
```
